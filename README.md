# GeoOS - Geospatial Operations System

A Next.js-based geospatial operations platform for real-time location analytics, demand forecasting, and driver optimization.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Real-time geospatial data visualization
- ML-powered demand forecasting
- Anomaly detection algorithms
- Driver rebalancing optimization
- Safety monitoring and recommendations
- Interactive heat maps and dashboards
